# Bank Snap

🔔 본 프로젝트는 **삼성 청년 SW 아카데미** 1학기 관통 프로젝트 결과물입니다

<br><br>



## **개요**

- 금융상품정보 웹 애플리케이션
- 예적금상품조회, 금융상품추천, 환율계산, 은행 검색, 커뮤니티 기능을 제공

<br><br>

## 프로젝트 기간

- 2023년 11월 14일 ~ 2023년 11월 24일 (11일)👋
